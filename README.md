# jots-web

React.js web client for jots

Related repositories:

-   [API](https://github.com/collinsmuriuki/jots-api)
-   [Desktop](https://github.com/collinsmuriuki/jots-desktop)
-   [Mobile](https://github.com/collinsmuriuki/jots-mobile)

## Usage

Install

```
npm install
```

Start

```
npm run dev
```
